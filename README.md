# 🛠️ ConvertKit Tool

ConvertKit is a lightweight and user-friendly web tool built with HTML, CSS, JavaScript, and PHP. It allows users to easily convert, compress, resize, and manage files such as images and PDFs directly from their browser.

📌 Features
Image Tools
Convert images between different formats (e.g., JPG, PNG, GIF, WebP)
Compress images to reduce file size
Resize images to custom dimensions
PDF Tools
Convert files to PDF and from PDF to other formats
Compress PDFs for smaller size
Merge multiple PDF files into a single document
Other File Tools
Convert files between supported formats
Compress files to save storage space
💻 Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: PHP
File Handling: Server-side file processing using PHP
⚡ How It Works
Upload your file (image, PDF, or document)
Choose the desired action (convert, compress, resize, merge)
Download the processed file

All processing is done on the server using PHP for speed and compatibility.
